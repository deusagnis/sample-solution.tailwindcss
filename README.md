# Sample solution on TailwindCSS
## Dependencies
Use command to install dependencies:

``
npm install
``

## Usage
Use command to build in dev:

``
npm run dev
``

Use command to watch in dev:

``
npm run watch
``

Use command to build in prod:

``
npm run prod
``
